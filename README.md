# 二次元笑传之抽抽Bot（现实人物版）

基于 NapCat 与 AstrBot 的 QQ 群聊集卡插件，卡池为 **Bangumi 现实人物** 数据（`util/现实人物.json`）。玩法类似 Mudae：抽人、结缘、老婆团、许愿人等。

## 部署

NapCat 与 AstrBot 请参考 [https://docs.astrbot.app/deploy/platform/aiocqhttp/napcat.html](https://docs.astrbot.app/deploy/platform/aiocqhttp/napcat.html)

## 参数设置

在「插件 → 插件配置」中编辑默认参数（如每周期抽人次数、结缘冷却、老婆名额上限等）。群内可用 **`抽人设置`** 覆盖本群配置。

## 基本指令

**查看帮助**

```
@机器人 抽人菜单
@机器人 抽人帮助
```

**抽人**

```
@机器人 抽人
@机器人 cr
```

**查看老婆团**

```
@机器人 我的老婆
@机器人 我的老婆 <页码>
```

**老婆排行**

```
@机器人 老婆排行
```

按本群老婆团总人气排序，展示收集进度。

**结缘**

给抽人结果消息贴表情即可收集该人物；也可回复该条消息「@机器人 **结缘**」。默认结缘冷却见插件配置。

**解除婚姻关系**

```
@机器人 解除婚姻关系 <人物ID>
```

**置顶老婆**

```
@机器人 置顶老婆 <人物ID>
```

置顶老婆在「清理老婆」和「人脉轮回」时会被保留。

**查人 / 搜人**

```
@机器人 查人 <人物ID>
@机器人 查人 <人物ID> <图片序号>
@机器人 搜人 <名称或部分名称>
```

图片序号为 1 到该人物图片总数（含自定义照片）；不填则随机一张。

**自定义照片**（结缘后可添加/清除，抽人与查人会加入图片池）

```
@机器人 加照片 <人物ID>
```

回复一张图片或随消息发送图片即可添加。每人每 ID 上限见插件配置。

```
@机器人 清照片 <人物ID>
```

结缘用户本人或群管理员/群主可清除该人物全部自定义照片。

**许愿人**

```
@机器人 许愿人 <人物ID>
@机器人 愿望单人
@机器人 删除许愿人 <人物ID>
```

作用：抽到时会 @许愿者、略增抽到概率、人气按许愿人数加成、并影响被牛概率等。

**换老婆**

```
@机器人 换老婆 <我的人物ID> <对方人物ID>
```

对方给该条换老婆请求消息贴表情即可完成交换。

## 管理员指令

**抽人设置**（子命令见机器人回复的无参说明，含抽人冷却、查人冷却、配额周期、允许时段、老婆上限、抽人范围、牛概率等；亦兼容旧关键词如「抽卡冷却」「后宫上限」「人脉上限」等）

```
@机器人 抽人设置
@机器人 抽人设置 <功能> <参数>
```

**清理老婆**

```
@机器人 清理老婆 <QQ号>
```

**强制解除**（解除某人物的结缘关系，修正异常数据）

```
@机器人 强制解除 <人物ID>
```

## 群主 / 超管指令

**人刷新**（重置该用户抽人配额与结缘冷却）

```
@机器人 人刷新 <QQ号>
```

**人脉轮回**（近似重开：清除本群结缘数据，**置顶老婆**保留）

```
@机器人 人脉轮回 确认
```
